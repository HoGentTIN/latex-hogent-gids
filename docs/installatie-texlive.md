## Installatie

=== "Windows"

    ```console
    choco install texlive --params="'/collections:xetex'"
    ```

=== "Debian/Ubuntu"

    ```console
    sudo apt install texlive-base texlive-latex-base texlive-latex-recommended texlive-bibtex-extra texlive-pictures texlive-fonts-recommended
    ```

=== "RedHat/Fedora/SuSE"

    ```console
    sudo dnf install 
    ```
